# Dokumen PPID - Informasi Tersedia Setiap Saat

Folder ini berisi dokumen-dokumen informasi yang tersedia setiap saat dan dapat diakses publik.

## Struktur File yang Dibutuhkan:

Letakkan file PDF dokumen dengan nama sebagai berikut:

1. `daftar-informasi-publik.pdf` - Daftar Informasi Publik
2. `laporan-penerima-bantuan-rtlh-2024.pdf` - Laporan Penerima Bantuan Perbaikan RTLH 2024
3. `sop-peringatan-dini-darurat.pdf` - SOP Peringatan Dini dan Keadaan Darurat Dinas Perkimtan
4. `sop-standar-pengujian-konsekuensi.pdf` - SOP Standar Pengujian Konsekuensi
5. `sop-standar-maklumat-pelayanan.pdf` - SOP Standar Maklumat Pelayanan
6. `sop-standar-pendokumentasian.pdf` - SOP Standar Pendokumentasian
7. `sop-penyusunan-daftar-informasi-publik.pdf` - SOP Penyusunan Daftar Informasi Publik
8. `sop-standar-penetapan-pemutakhiran.pdf` - SOP Standar Penetapan dan Pemutakhiran Daftar informasi
9. `sop-standar-pengajuan-keberatan.pdf` - SOP Standar Pengajuan keberatan
10. `sop-standar-pengumuman-informasi-publik.pdf` - SOP Standar Pengumumuman Informasi Publik
11. `pedoman-teknis-pembangunan-perumahan.pdf` - Pedoman Teknis Pembangunan Perumahan
12. `regulasi-pengelolaan-kawasan.pdf` - Regulasi Pengelolaan Kawasan Permukiman
13. `prosedur-pelayanan-pertanahan.pdf` - Prosedur Pelayanan Pertanahan
14. `standar-biaya-masukan-2025.pdf` - Standar Biaya Masukan Tahun 2025
15. `formulir-permohonan-informasi.pdf` - Formulir Permohonan Informasi Publik

## Cara Menggunakan:

1. Letakkan file PDF dokumen di folder ini
2. Pastikan nama file sesuai dengan yang tercantum di atas
3. File akan otomatis dapat diakses melalui halaman Informasi Setiap Saat
4. Link "Lihat File" akan membuka dokumen di tab baru

## Catatan:

- Format file: PDF
- Pastikan file tidak terlalu besar (maksimal 10MB per file untuk performa optimal)
- Nama file harus sesuai dengan yang sudah ditentukan agar link berfungsi dengan benar
