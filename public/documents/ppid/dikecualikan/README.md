# Dokumen PPID - Informasi Dikecualikan

Folder ini berisi dokumen-dokumen terkait informasi yang dikecualikan dari keterbukaan informasi publik.

## Struktur File yang Dibutuhkan:

Letakkan file PDF dokumen dengan nama sebagai berikut:

1. `daftar-informasi-dikecualikan-2025.pdf` - Daftar Informasi yang Dikecualikan Tahun 2025
2. `pedoman-klasifikasi-rahasia.pdf` - Pedoman Pengklasifikasian Informasi Rahasia
3. `sk-penetapan-dikecualikan-2025.pdf` - SK Penetapan Informasi yang Dikecualikan 2025
4. `konsekuensi-pengujian-rahasia.pdf` - Konsekuensi Pengujian Informasi Rahasia
5. `tata-cara-uji-konsekuensi.pdf` - Tata Cara Permohonan Uji Konsekuensi
6. `penjelasan-klasifikasi-terbatas.pdf` - Penjelasan Klasifikasi Informasi Terbatas
7. `mekanisme-pembukaan-rahasia.pdf` - Mekanisme Pembukaan Informasi Rahasia
8. `dasar-hukum-pengecualian.pdf` - Dasar Hukum Pengecualian Informasi Publik

## Cara Menggunakan:

1. Letakkan file PDF dokumen di folder ini
2. Pastikan nama file sesuai dengan yang tercantum di atas
3. File akan otomatis dapat diakses melalui halaman Informasi Dikecualikan
4. Link "Lihat File" akan membuka dokumen di tab baru

## Catatan:

- Format file: PDF
- Informasi dikecualikan sesuai UU No. 14 Tahun 2008 tentang Keterbukaan Informasi Publik
- Dokumen berisi penjelasan tentang informasi yang tidak dapat diakses publik dan alasannya
- Terdapat mekanisme uji konsekuensi bagi publik yang keberatan
- Nama file harus sesuai dengan yang sudah ditentukan agar link berfungsi dengan benar
