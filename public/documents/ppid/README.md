# Dokumen PPID - Informasi Berkala

Folder ini berisi dokumen-dokumen informasi berkala PPID yang dapat diakses publik.

## Struktur File yang Dibutuhkan:

Letakkan file PDF dokumen dengan nama sebagai berikut:

1. `lkjip-2024.pdf` - LKjIP 2024
2. `rekap-dpa-2025.pdf` - Rekap DPA 2025
3. `laporan-layanan-informasi-2024.pdf` - Laporan Layanan Informasi Perkimtan 2024
4. `laporan-keuangan-2024.pdf` - Laporan Keuangan 2024
5. `struktur-organisasi-2025.pdf` - Struktur Organisasi Dinas Perkimtan 2025
6. `data-lokasi-psu-2025.pdf` - Data Lokasi PSU Per Jumlah Kecamatan Tahun 2025
7. `rkt-2025.pdf` - RKT 2025
8. `perjanjian-kinerja-eselon-ii.pdf` - Perjanjian Kinerja Eselon II
9. `realisasi-program-mei-2025.pdf` - Realisasi Program Kegiatan sd Mei 2025
10. `kak-penyusunan-dokumen-2025.pdf` - KAK Penyusunan Dokumen Perangkat Daerah 2025
11. `renstra-2024-2026.pdf` - Rencana Strategis 2024-2026
12. `lakip-2024.pdf` - Laporan Akuntabilitas Kinerja 2024
13. `profil-dinas-2025.pdf` - Profil Dinas Perkimtan 2025
14. `data-rtlh-2024.pdf` - Data Rumah Tidak Layak Huni 2024
15. `rkt-2024.pdf` - Rencana Kerja Tahunan 2024

## Cara Menggunakan:

1. Letakkan file PDF dokumen di folder ini
2. Pastikan nama file sesuai dengan yang tercantum di atas
3. File akan otomatis dapat diakses melalui halaman Informasi Berkala
4. Link "Lihat File" akan membuka dokumen di tab baru

## Catatan:

- Format file: PDF
- Pastikan file tidak terlalu besar (maksimal 10MB per file untuk performa optimal)
- Nama file harus sesuai dengan yang sudah ditentukan agar link berfungsi dengan benar
