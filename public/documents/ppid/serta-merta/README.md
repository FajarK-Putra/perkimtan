# Dokumen PPID - Informasi Serta Merta

Folder ini berisi dokumen-dokumen informasi serta merta yang wajib diumumkan tanpa penundaan terkait keadaan darurat atau bencana.

## Struktur File yang Dibutuhkan:

Letakkan file PDF dokumen dengan nama sebagai berikut:

1. `pengumuman-bencana-banjir-padang.pdf` - Pengumuman Bencana Alam Banjir Wilayah Padang
2. `peringatan-dini-longsor.pdf` - Peringatan Dini Potensi Longsor Area Permukiman
3. `info-darurat-kerusakan-infrastruktur.pdf` - Informasi Darurat Kerusakan Infrastruktur Perumahan
4. `pengumuman-evakuasi-permukiman.pdf` - Pengumuman Evakuasi Permukiman Daerah Rawan Bencana
5. `peringatan-kebakaran-lahan.pdf` - Peringatan Ancaman Kebakaran Lahan Permukiman
6. `info-tanggap-darurat-gempa.pdf` - Info Tanggap Darurat Gempa Bumi Sumatera Barat
7. `pengumuman-perbaikan-darurat.pdf` - Pengumuman Perbaikan Darurat Fasilitas Umum
8. `peringatan-cuaca-ekstrem.pdf` - Peringatan Cuaca Ekstrem Dampak Permukiman
9. `info-darurat-pencemaran-air.pdf` - Info Keadaan Darurat Pencemaran Air Bersih
10. `pengumuman-bahaya-struktur-bangunan.pdf` - Pengumuman Bahaya Struktur Bangunan Tidak Layak Huni

## Cara Menggunakan:

1. Letakkan file PDF dokumen di folder ini
2. Pastikan nama file sesuai dengan yang tercantum di atas
3. File akan otomatis dapat diakses melalui halaman Informasi Serta Merta
4. Link "Lihat File" akan membuka dokumen di tab baru

## Catatan:

- Format file: PDF
- Informasi serta merta bersifat darurat dan harus segera diumumkan
- Pastikan dokumen berisi informasi yang jelas dan akurat
- Update secara berkala sesuai situasi darurat yang terjadi
- Nama file harus sesuai dengan yang sudah ditentukan agar link berfungsi dengan benar
